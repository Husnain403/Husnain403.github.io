Demographic Analytics Dashboard

Team Members: Husnain, Ali, Sara, Ahmed

Project Description

This project is a web-based dashboard that visualizes demographic data for countries around the world. The dashboard provides insights into population trends, annual growth rates, and global population distributions with interactive charts and maps.

Dataset Description

The dataset used is countries_population.csv, which contains:

  • Country Name
  • Year
  • Population
  • Annual Growth Rate (%)

The dataset covers multiple years and includes global population statistics for all countries.

Tools and Technologies Used

  • HTML, CSS, JavaScript
  • Chart.js for interactive charts
  • PapaParse for CSV parsing
  • Datamaps (D3.js) for world map visualization
  • GitHub Pages for hosting the webpage
